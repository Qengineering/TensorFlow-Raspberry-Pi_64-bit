### Ubuntu 18.04
TensorFlow 2.5.0 depends on h5py version 3.1.0. Unfortunately, the h5py version 3.1.0 cannot be easily installed on Ubuntu 18.04, or to be more precise, on an aarch64 with Python 3.6. See [#1760](https://github.com/h5py/h5py/issues/1760). That's why we don't have a wheel for Ubuntu 18.04. Use TensorFlow 2.4.1 or switch to Ubuntu 20.04.<br/><br/>
